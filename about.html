This application was built with Electron. Electron uses HTML, CSS, and Javascript.

Electron (formerly known as Atom Shell) came about in 2013 and was developed by a sole developer before being taken on by GitHub. Many popular applications such as Slack, GitHub Desktop, and Discord were built with the Electron framework.
